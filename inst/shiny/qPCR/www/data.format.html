Please export Ct values from your qPCR machine/software.

    Ct values should have be labeled with well index, gene name, sample tags and repeat number.
  1. save data in csv or tab-delim format
  2. required columns: "Well", "Ct", "gene" and "rep.num"
  3. sample labels (required) can be put in a single "sample" column, or presented in multi columns named with "ds.xxxx" pattern.
  4. duplicated well label ("Well" column) are not allowed
  5. all samples should have tested same list of genes of identical repeats.
  6. avoid using space and other special character in column names.
